Amiibo Finder

Search by Characters Search by Game Series Documentation

Process

Going into the project I tried to base elements of the Giphy Finder and the sketch design that I wanted to do. I wanted to use the Amiibo figurines itself to create a background and a headline for the project.

When designing the site I had to make a lot of modifications in order to get the API to display. Therefore, my design changed from what I initially wanted to design. I put in all the backgrounds, header background, and gave a gradient background for the Amiibo type to be displayed.

When having the my milestone checkpoint, I did not do much besides making changes to the backgrounds. The API also was not displaying correctly at the time either. It still looked like the Giphy finder. Also, the font I wanted to use wasn't working properly.

My final process was to make drastic modification. I was able to get my font to work, in which I realized the font needed to be in the same folder as the style sheet. My API also worked properly, so that I could display the data that I wanted to. I also rechanged my background image so it wouldt have the same background as the image result background.

I also saved the local storage information, so when switching between “tabs” it keeps the last saved data. The saved local storage will also work when closing the browser as well.
